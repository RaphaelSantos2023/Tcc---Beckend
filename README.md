# **Pré-requisitos** 
Antes de iniciar o projeto, certifique-se de ter instalado os seguintes softwares:
- Python 3.10+
- Node.js e npm (npm já vem junto com o Node.js)
- Ter chave para gemini, url supabase e chave supabase

# **Bibliotecas do python** 
Blibiotecas utilizadas e necessarias para  api rest funcionar:
pip install flask mysql-connector-python passlib pyjwt flask-cors supabase

# **Rodar o front temporario** 
Tem que:
- Baixar node.js
- ir, pelo terminal, ao diretorio da pasta frontend
- digite *npm start* no terminal

# **Rodar o back temporario** 
Tem que:
- ir, pelo terminal, ao diretorio da pasta backend
- digite *python app.py* no terminal
